What is grid based design?

A grid is usually a structure made up of vertical and horizontal lines that intersect, used to structure content.

For example, this page uses a grid structure that I've changed fit my desires. Grids are used in various software such as Microsoft Excel, and used in applications such as video games.

Whats all the hype about responsive webpages?

Being responsive means that the page will conform to the hardware that a user is using. This ensures that no matter the hardware (phone, tablet, pc or mac) the website will function.

Try to change the width of the page. The page should conform properly.

What is semantic structure?

Html uses a set of tags that are priotized and has different purposes.

For example: h1 and h4 has a different font size, which is useful for presentation.
